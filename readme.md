# laravel-vue-cms

## clone & install dependencies

```
clone https://github.com/liukaijv/laravel-vue-cms.git someDir

cd someDir

composer install

npm install 

```

## config

open .env file & config database 

```
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel-vue-cms
DB_USERNAME=root
DB_PASSWORD=

```

## run 

```
npm run web
npm run dev

```
